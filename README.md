# FurriBot — Telegram бот с AI фурри-персонажем

Простой, но многофункциональный бот с искусственным интеллектом, социальной сетью и RPG-элементами. Говорит "ня", "мяу" и имеет настроение.

## Возможности

- Общение с AI (OpenAI или совместимый API)
- Профили пользователей (био, теги)
- Группы по интересам (создание/вступление)
- Система уровней и опыта
- Ежедневные награды
- Инвентарь с предметами
- Квесты
- Интерактивные команды (/hug, /pat, /purr)
- Случайная смена настроения персонажа

## Установка и запуск

### 1. Получить токен бота

1. Найди в Telegram [@BotFather](https://t.me/botfather)
2. Отправь команду `/newbot`
3. Следуй инструкциям, получи токен (выглядит как `123456:ABCdefGH...`)

### 2. Получить ключ AI API

Если используешь OpenAI:
- Зарегистрируйся на [OpenAI](https://platform.openai.com)
- Создай API ключ в разделе API keys

Если используешь другой API (совместимый с OpenAI), укажи его в `AI_BASE_URL`.

### 3. Подготовка окружения

Склонируй репозиторий или создай папку проекта со всеми файлами из этой инструкции.

Создай файл `.env` на основе `.env.example` и вставь свои ключи:

BOT_TOKEN=твой_токен
AI_API_KEY=твой_ai_ключ

### 4. Установка зависимостей

Убедись, что у тебя установлен Python 3.11 или новее. Затем выполни:
